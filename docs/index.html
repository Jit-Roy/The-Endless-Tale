Getting Started

1. Prerequisites

  • Python 3.10+
  • Google Gemini API key

2. Installation

git clone https://github.com/Jit-Roy/The-Endless-Tale.git
cd "The Endless Tale"
pip install -r requirements.txt

3. Environment Variables

Create a .env file in the root directory and add your Google Gemini API keys:

GOOGLE_API_KEY=your_primary_api_key_here
BACKGROUND_GOOGLE_API_KEY=your_background_api_key_here

4. Run the Engine

python app.py

Creating Custom Stories & Characters

You can easily build your own dynamic adventures by defining a story directory with JSON files.

{
    "title": "Space Odyssey",
    "description": "A journey to the stars.",
    "objectives": [
        "Wake up from cryo-sleep",
        "Fix the main engine",
        "Navigate the asteroid field"
    ]
}
Performance Note: Because the system manages an intricate, dynamic world—evaluating heavy narrative context, tracking objectives, and generating parallel decisions—API calls are computationally intensive. On average, it takes about 40 seconds for a new message to appear in the UI. Please be patient while the engine computes the next phase!
